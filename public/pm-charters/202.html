Objective
Build a longitudinal cognitive performance measurement system combining standardized testing, chess ELO tracking, and Neural NeXus game analytics — with supplement/intervention phase markers — to objectively evaluate whether cognitive enhancement strategies are working.

Three-Track Measurement System
TrackPurposeFrequencyNormalization
Standardized BatteryControlled signal, same difficulty every time3x/weekSelf-normalizing (same test)
Chess ELOReal-world composite cognitive metric (passive)Ongoing (natural play)Self-normalizing (rating system)
Neural NeXus GamesSupplementary, ecological validityDaily (natural play)14-day rolling average
Standardized Battery (~3 min total)
Dual N-back (90s) — working memory, fluid intelligence: max N level, accuracy %  |  PVT (60s) — reaction time, sustained attention: median RT, lapses >500ms  |  DSST (60s) — processing speed: correct substitutions/min
✅ In Scope
  • Web-based standardized cognitive tests (dual n-back, PVT, DSST)
  • Chess.com API integration (ELO, accuracy, centipawn loss)
  • Server-side game result logging (Spelling Bee, Wordle, Crossword)
  • Trends dashboard with phase markers (vertical lines at interventions)
  • Simple auth (password/token) — private dashboard
  • Statistical helper: effect size, confidence intervals
  • Data export (CSV)
  • 2-week baseline before any intervention
🚫 Out of Scope
  • Mobile native app (web-only)
  • EEG / HRV / hardware measurements
  • Automated supplement recommendations (Charter 201)
  • Social / multiplayer features
