Objective
Build a daily Wordle word-guessing game for the Neural NeXus website. Same premium dark UI as Spelling Bee. New word every day, 6 guesses, color-coded feedback, shareable results.
Success Criteria
Playable at www.neuralnexus.press/wordle, daily word rotation, keyboard with color hints, share-friendly emoji grid, stats tracking.

Design — Architecture
Browser (player) ↕ Static Frontend (wordle.html — game UI, keyboard, stats) ↕ REST API (Express) ├── GET /api/wordle/today → today's date + word length └── POST /api/wordle/guess → validate guess, return colors ↕ wordle-words.js ├── ANSWERS[] — ~850 curated common words ├── dict5.txt — 8,500 valid guesses └── getTodayWord() — deterministic daily pick
Design — Color System
🟩 Green — correct letter, correct position
🟨 Yellow — correct letter, wrong position
⬛ Gray — letter not in the word
Keyboard updates cumulatively (best state wins)
Design — Stats Model
localStorage['wordle-stats'] ├── played (total games) ├── won (total wins) ├── streak (current) ├── maxStreak (all-time) └── lastDate (for streak calc) localStorage['wordle-state-YYYY-MM-DD'] ├── guesses[] (words + results) ├── gameOver (bool) └── won (bool)
