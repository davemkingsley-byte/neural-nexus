Dual N-Back

A demanding working memory task that trains two channels simultaneously. On each trial you see a position on a grid and hear a letter. Your job is to identify when either the position or the sound matches what appeared N trials ago. The difficulty adapts as you play — the better you do, the higher N climbs.

How it works:
• A square lights up on a 3×3 grid — press A or tap Position Match if it matches N steps back.
• A letter is spoken — press L or tap Sound Match if it matches N steps back.
• You start at 2-back. Three correct in a row levels you up; three wrong drops you down.
• 20 trials. Stay sharp.


How It Works

What is N-Back?

The N-Back task is one of the most well-studied working memory tests in cognitive science. You must continuously compare what you see and hear now with what happened N steps ago. As you improve, N increases — making the task progressively harder.

1. Visual Channel — Watch the grid

Each turn, a square lights up on the 3×3 grid. Remember where it appears. If the position matches the one from N turns ago, press A or tap Position Match.

2. Audio Channel — Listen for letters

A letter is spoken each turn. When the current letter matches the one from N turns ago, press L or tap Sound Match.

3. Dual Task — Both at once

The real challenge: you must track both channels simultaneously. A position match and an audio match can happen on the same trial — or neither.

4. Scoring & Adaptive Difficulty

You start at 2-back. Three correct trials in a row raises N by one; three wrong drops it. Your key metric is Max N — the highest level you reach. Visual and audio accuracy are tracked separately.
