How It Works

What is N-Back?

A gold-standard working memory test. You compare what you see and hear now with what happened N steps ago. As you improve, N increases.

1. Visual — Watch the grid

A blue square lights up in the 3×3 grid each turn. If it matches the position from N turns ago, tap Position Match.

2. Audio — Listen for letters

A letter is spoken each turn. When it matches N steps ago, tap Audio Match.

3. Both at once

Track both channels simultaneously. Position and audio matches can happen on the same trial — or neither.

4. Scoring

Start at 2-back. Accuracy above 80% raises N; below 50% lowers it. Your key metric is Max N reached. Visual and audio accuracy are tracked separately.
