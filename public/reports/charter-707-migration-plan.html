📋 Executive Summary

Anthropic is revoking third-party API access for Claude Max subscriptions effective April 4, 2026 at 2:00 PM CT. This affects all four Opus-based agents in our fleet. Without action, the entire C-suite AI infrastructure goes dark tomorrow afternoon.

Running the current fleet unconstrained on API billing would cost approximately $3,780/mo — 4–6× over budget. This plan restructures the fleet to hit a target of $600–810/mo through model downgrades, delegation to free sub-agents, local model heartbeats, and aggressive cost controls.

Subscription savings from cancelling ChatGPT Pro and Claude Max #2 free up $400/mo, combined with $400/mo new budget allocation for a total API envelope of $600–1,000/mo.

7 decisions require David's approval before migration can proceed. Phase 0 (cost controls) begins tonight. Billing setup is needed from David by tomorrow morning.

1 The Problem

Our agent fleet was built on flat-rate Claude Max subscriptions ($200/mo each × 2 accounts = $400/mo) providing unlimited Opus API access. Anthropic is ending this for high-usage accounts, forcing a transition to pay-per-token billing where every API call costs real money.

Critical insight: Every message to an Opus agent sends the entire conversation history as input tokens. A session at 100K context costs ~$1.50 per message in input alone — and it compounds with every turn. This is why an hour of casual use cost $20-25 on API billing.
